About SMI-BLAST

PSI-BLAST is a mainstream iterative search tool for protein sequence searching. The generation of accurate Position-Specific Scoring Matrix (PSSM) is the key to improve its performance. For protein remote homology detection, more non-homologous sequences can be detected when PSI-BLAST selects non-homologous sequence to construct PSSM in the iterative process.

In this study, three types of Incorrectly Selected Homology (ISH) errors (?) of PSSM are explored on SCOPe benchmark dataset, and a corresponding solution called Sequence similarity Matrix-based Iterative BLAST (SMI-BLAST) is proposed. Furthermore, SMI-HMMER is proposed by completely applying SMI-based framework on JackHMMER with default iteration

SMI-BLAST and SMI-HMMER are trained with an updated benchmark dataset SCOPe v2.06 with more samples and protein families, making it more robust and useful. The web server will be updated when new version of SCOPe is released.
